Bem-vindo ao Análise de Dados COVID-19

Este projeto visa analisar dados sobre a pandemia de COVID-19, fornecendo insights importantes através de gráficos interativos.

Problema

A pandemia de COVID-19 trouxe desafios sem precedentes para o mundo. Monitorar e analisar os dados de casos confirmados, mortes e recuperações é crucial para entender a evolução da doença.

Solução Proposta

Utilizando ferramentas como Pandas para manipulação de dados, Matplotlib para geração de gráficos e Flask para desenvolvimento web, este projeto cria um ambiente interativo para visualização dos dados.

Base de Dados Utilizada

Os dados utilizados neste projeto foram obtidos de COVID-19 Dataset - Kaggle, uma fonte confiável e atualizada.

Contexto do Dataset

Um novo coronavírus denominado 2019-nCoV foi identificado pela primeira vez em Wuhan, capital da província de Hubei, na China. Pessoas desenvolveram pneumonia sem uma causa clara e para a qual as vacinas ou tratamentos existentes não eram eficazes. O vírus demonstrou evidências de transmissão de pessoa para pessoa. A taxa de transmissão (taxa de infecção) pareceu aumentar em meados de janeiro de 2020. Em 30 de janeiro de 2020, aproximadamente 8.243 casos foram confirmados.
